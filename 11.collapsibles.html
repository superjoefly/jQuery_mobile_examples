jQuery Mobile - Collapsibles

Collapsible Content Block

Collapsibles allows you to hide or show content, which is useful for storing parts of information.

To create a collapsible block of content, assign the data-role="collapsible" attribute to a container. Inside the container (div), and a header (h2 etc.) or legend element, followed by any HTML markup you want to be expanded.

Example: collapsible

Click Me...I'm collapsible!

I'm the collapsible/expanded content...

By default the content is closed. To expand the content when the page loads, use data-collapsed="false".

Example: data-collapsed="false"

Click me...I'm collapsible!

I'm now expanded by default.

Nested Collapsible Blocks

Collapsible content blocks can be nested (a collapsible inside a collapsible).

Example: nested collapsible

Click me...I'm collapsible!

This is some text!!!

This is a nested collapsible div!

Wow! That's awesome!

Note: collapsible content blocks can be nested as many times as you want.

Collapsible Sets

Collapsible sets are collapsible blocks that are grouped together (often referred to as an accordion). When a new block is opened, all other blocks close.

Create several collapsible content blocks, and then wrap a new container with the data-role="collapsibleset" arround the collapsible blocks.
