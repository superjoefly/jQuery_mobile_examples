jQuery Mobile - Getting Started

Adding jQuery Mobile To Your Web Pages

There are two ways to add jQuery Mobile to your website:

  • Link to CDN (recommended)
  • Link to library stored on computer/site

Link to CDN

A CDN (Content Delivery Network) is used to distribute often-used files across the web. This makes download speed much faster for the user.

You don't have to install anything on your computer: just include the following:

  1. /meta name="viewport" content="width=device-width, initial-scale=1">
  2. /link rel="stylesheet" href="https://code.jquery.com/mobile/1.4.5/jquery.mobile-1.4.5.min.css">
  3. /script src="https://code.jquery.com/jquery-1.11.3.min.js">
  4. /script src="https://code.jquery.com/mobile/1.4.5/jquery.mobile-1.4.5.min.js">

Link to Locally Stored Library

If you want to host the library yourself, you must first download it from jQuerymobile.com.

Place the downloaded files in the same directory as the pages where you wish to use it.

Then add the following:

  1. /meta name="viewport" content="width=device-width, initial-scale=1">
  2. /link rel="stylesheet" href="jquery.mobile-1.4.5.css">
  3. /script src="jquery.js">
  4. /script src="jquery.mobile-1.4.5.js">
