jQuery Mobile - Events

Events = All the different visitor actions that a webpage can respond to.

jQuery Mobile Events

You can use any standard jQuery Events in jQuery Mobile.

In addition, jQuery Mobile offers several events that are tailor made for mobile browsing.

  • Touch events - when user taps and swipes
  • Scroll events - when user scrolls up or down
  • Orientation events - when device is rotated vertically or horizontally
  • Page events - when page is shown, hidden, created, loaded and/or unloaded

See reference for more complete list.

Initializing jQuery Mobile Events

jQuery Mobile uses a pagecreate event, which occurs when a page has been created in the DOM, yet before enhancement is complete, to prevent code from running pre-maturely.

The second parameter ("#pageone") points to the id of the page to specify the events for:

Example: pagecreate event


						$(document).on("pagecreate", "#pageone", function() {
							//jQuery Events Go Here
						})

Note: the jQuery on() method is used to attach event handlers.
