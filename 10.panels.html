jQuery Mobile - Panels

Panels in jQuery Mobile will slide out from the left or the right side of the screen with additional content.

To create a panel, add the data-role="panel" attribute to a div-element and specify an id.

Add any HTML markup inside this div-element that you want to display in you panel.

Example: panel

Note: the panel markup must be placed before or after the header and footer content, inside a jQuery Mobile page.

To access the panel, create a link that points to the id of the panel div-element. When a user clicks on the link, the panel will open.

Closing Panels

You can close the panel by clicking outside of it, by swiping or by pressing the Esc key. You can disable the clicking and swiping features by adding data-* attributes to the panel div-element.

Attribute Value Description
data-dismissible true | false specifies whether the panel can be closed by clicking outside of it or not.
data-swipe-close true | false specifies whether the panel can be closed by swiping or not

You can also close the panel by using a button: Add a link inside the panel div with the data-rel="close" attribute attached to it. And for compatibility reasons, you should also specify the href attribute to point to the ID of the page the user should go to when closing the panel.

Panel Display

You can control the display mode of the panel with the data-display attribute.

Attribute Value Description
data-display="overlay" displays the panel over the content
data-dispaly="push" animates both the panel and the page at the same time (pushes page over)
data-display="reveal" dafault; the panel will sit under the page and reveal as the page slides away.

Positioning Panels

By default panels will appear on the left side of the screen. For a panel to appear on the right side, specify the data-position="right" attribute.
