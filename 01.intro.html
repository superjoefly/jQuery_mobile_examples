jQuery Mobile - Intro

jQuery Mobile is a web framework for creating mobile web applications.

Why Use jQuery Mobile?

jQuery is a touch-optimized web framework for creating mobile web applications.

jQuery Mobile works on all popular smartphones and tablets.

jQuery is built on top of the jQuery library, which makes it easy to learn if you already know jQuery.

It uses HTML5, CSS3, JavaScript and AJAX to accomplish its work for laying out pages with minimal scripting.

jQuery is a "write less, do more" framework: it automatically designs web pages with an attractive and "easy-to-use" look, that will work the same way on all mobile devices.

Instead of writing one application for each mobile device or OS, jQuery Mobile allows you to only write one application. It only uses HTML, CSS and JavaScript, which is standard for all mobile web browsers.

Best Reading Experience

Note: Even though jQuery Mobile works on all mobile devices, it may have some compatibility issues on desktop computers (due to limited CSS3 support).
