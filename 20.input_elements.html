jQuery Mobile Text Inputs

Input fields are coded with standard HTML elements, and jQuery Mobile will style them to look attractive and easy-to-use for mobile devices. You can also use the HTML5 input-types:

Text Area

Use "textarea" for multi-line text inputs.

Note: The text area will automatically grow to fit new lines as you type some text.

Example: text area

Search Input

The input type="search" is new in HTML5, and defines a text field for entering a search.

Radio Buttons

Radio buttons are used when a user can select only one of a limited number of choices.

To create a set of radio buttons, add an input with type="radio" and a cooresponding label. Wrap the radio buttons in a fieldset element. You can also add a legend element to define a cation for the fieldset.

Tip: Use the data-role="controlgroup" to group the buttons together.

Checkboxes

Checkboxes are used when a user can select one or more options of a limited number of choices.

More options:

To group radio buttons or checkboxes horizontally, use the data-type="horizontal" attribute.

Example code: horizontal buttons


		/fieldset data-role="controlgroup" data-type="horizontal"/

You can also wrap a field container around the fieldset element.

Example: ui-field-contain


		/div class="ui-field-container"/
			/fieldset data-role="controlgroup"/
				/legend/ Chose your gender.../legend/
			/fieldset
		/div

If you want one of the buttons to be pre-selected, use the HTML input "checked" attribute:


		/input type="radio" checked /
		/input type="checkbox" checked /
