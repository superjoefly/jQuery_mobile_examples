jQuery Mobile List Icons

The default icon for each list item containing a link is a "carat-r" (right arrow). To change this to another icon, use the data-icon attribute on the list item you want to modifiy.

Example: changing list Icons

  • Default is right arrow.
  • data-icon="plus"
  • data-icon="minus"
  • data-icon="delete"
  • data-icon="location"
  • data-icon="false"

Note: data-icon="false" will remove the icon.

Split Buttons

To create a split list with a vertical divider bar, place two links inside the li-element.

jQuery Mobile will automatically place the second link on the right side of the list with a right arrow-icon. The text inside the link, if any, will show when the user hovers over the icon.

Count Bubbles

Count bubbles are used to display numbers associated with list items, such as messages in a mailbox.

To add count bubbles, use an inline element, such as span, with a class of ui-li-count and add a number:

Example:

  • Inbox25
  • Sent432
  • Trash7

Note: to show the correct number in a count bubble, it must be updated programmatically. Explained in later chapter.
