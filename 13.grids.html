jQuery Mobile - Layout Grids

jQuery Mobile provides a set of CSS-based column layouts. However, column layouts are not generally recommended on a mobile device, due to the mobile's screen width.

But, there are times you may want to poisiton smaller elements, like buttons or nav tabs, side-by-side, as if it were in a table. Then, columns are perfect.

Columns in a grid are of equal width (and 100% wide in total), with no border, background, margin or padding.

There are 5 layout grids that can be used:

Grid Class Columns Column Widths (%) Corresponds To
ui-grid-solo 1 100 ui-block-a
ui-grid-a 2 50/50 ui-block-a|b
ui-grid-b 3 33/33/33 ui-block-a|b|c
ui-grid-c 4 25/25/25/25 ui-block-a|b|c|d
ui-grid-d 5 20/20/20/20/20 ui-block-a|b|c|d|e

Within the column container, child elements can have the class ui-block-a|b|c|d|e depending on the number of columns. The columns will float side by side.

Example 1: for the ui-grid-a class (which is a two column layout), you must specify two child elements: ui-block-a, ui-block-b.

Example 2: for ui-grid-b (three column layout), add three child elements: ui-block-a, ui-block-b, ui-block-c.

Customize Grids

You can customize your column blocks by using CSS.

Example: customize grid (see above example)

You can also customize blocks by using inline styles.

Multiple Rows

It is also possible to have multiple rows inside your columns.

Note: the ui-block-a class will always create a new line.

Example: adding a new row (ui-block-a) (see above example!)

Responsive Grids

On small screens, it is not recommended to have too many buttons with text side-by-side on one row - the text might get shortened.

For responsive grids, add the ui-responsive class to the container.
