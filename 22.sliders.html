jQuery Mobile - Form Sliders

A slider allows you to select a value from a range of numbers.

To create a slider, use input type="range"

Example: slider

Use the following attributes to specify restrictions:

  • max - maximum value allowed
  • min - minimum value allowed
  • step - legal number intervals
  • value - dafault value

Tip: If you'd rather show the value on the slider button, add data-show-value="true".

Tip: if you want the value to popup on your screen like a tooltip (as you slide), add data-popup-enabled="true".

Tip: if you want to highlight the track up to the slider value, add data-highlight="true".

Flip Toggle Switch

A flip switch is often used for on/off buttons:

To create a flip switch, use input type="checkbox" and specify a data-role="flipswitch".


By default the text on the flip switch is "On" and "Off". Use the data-on-text and data-off-text attributes to change this.


Use the "checked" attribute to set one of the options to be pre-selected.
