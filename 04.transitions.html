jQuery Mobile includes CSS3 effects that let you choose the way a page should open.

jQuery Mobile Transition Effects

jQuery Mobile has a variety of effects for how to transition from one page to the next.

Note: To achieve the transition effect, the browser must support CSS3 3D Transforms.

The transition effect can be applied to any link or form submission by using the data-transition attribute:


		/a href="#anylink" data-transition="slide">Slide to Page Two /a

The following available transitions can be used with the data-transition attribute for both pages and dialogs:

  • fade: default; fades to next page
  • flip: flips to next page from back to front
  • flow: throws the current page away and comes in with the next page
  • pop: goes to the next page like a pop-up window
  • slide: slides to the next page from right to left
  • slidefade: slides from right to left and fades in the next page
  • slideup: slides to the next page from bottom to top
  • slidedown: slides to the next page from top to bottom
  • turn: turns to the next page
  • none: no transition effect

The fading effect is default on all links in jQuery Mobile (if the browser supports it).

Tip: All effects above also support reverse/backward actions, e.g. if you want the page to slide from left to right, instead of right to left, use the data-direction attribute with value "reverse".
