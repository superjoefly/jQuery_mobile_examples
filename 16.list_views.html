jQuery Mobile - List Views

List views in jQuery Mobile are standard HTML list; ordered lists (ol), and unordered lists (ul).

To create a list, apply the data-role="listview" to the ol or the ul elements. To make the items tappable, specify a link inside each li-element.

By default, links inside a list item will automatically turn into a button (no need for ui-class="btn" or data-role="button").

List Dividers

List dividers are used to organize and group items into categories/sections.

To specify a list divider, add the data-role="list-divider" attribute to a li-element.

Example: list divider

  • Europe
  • Norway
  • Germany

If you have an alphabetical list, (for example a phonebook), jQuery Mobile automatically adds appropriate dividers by setting the data-autodividers="true" attribute on the ol or ul-element.

  • Adele
  • Agnes
  • Billy
  • Calvin

The data-autodividers="true" attribute creates dividers with uppercased first letters of the item's text.
